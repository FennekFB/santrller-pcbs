## 📁 Folder Hierarchy

```
santrller-projects/
├── boards/
│   ├── fusion 360/
│   │    └── "your user name"/
│   ├── kicad/
│   │    └── "your user name"/
│   ├── easyeda/
│   │    └── "your user name"/
│   └── proteus/
│       └── "your user name"/
├── gerbers/
│       └── "your user name"/
└── 3d models/
        └── "your user name"/
```

- Use the appropriate subfolder to upload your files.
- For community contributions, place your designs inside the `user made/` folder under your username.
